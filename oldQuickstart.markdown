# Quickstart

## Overview

In this guide, we will create a simple application which will
illustrate the following concepts...

* Static Lights
* Dynamic Lights
* Ambient Light
* Parallax

...we will also utilize the following types...

* Camera
* Engine
* EntityLayer
* TileLayer
* LightingModel
* Module
* Tile
* TileLayer
* ViewControl

This guide will walk through the creation of an auto-scrolling
environment split into a number of sections, each of which illustrates
a different feature...

1. The first section will have the ambient lighting turned up
bright and all other lights off.
2. The next section will turn off the ambient light, then turn
on a red, green, then blue light in succession.  This illustrates
dynamic lighting and shows how the lights mix with each other.
3. The third section will turn off all lights, then turn on an
off-white light surrounded by columns.
4. This section will turn off all lights, then turn on the token's
light.  The token will then move through a number of columns to show
the shadow casting capabilities.
5. The token will then move through a corridor and wrap back to the
beginning.

Throughout the environment, parallax will be utilized to give the walls
and pillars a retro pseudo-3d look.
